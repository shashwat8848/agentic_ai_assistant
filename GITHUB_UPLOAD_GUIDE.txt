=================================================================
  HOW TO UPLOAD YOUR PROJECT TO GITHUB (Step-by-Step Guide)
=================================================================

STEP 1 — Create a GitHub Account (if you don't have one)
  → Go to: https://github.com
  → Click "Sign Up" and create a free account

STEP 2 — Create a New Repository
  1. After logging in, click the "+" button (top right corner)
  2. Select "New repository"
  3. Repository name: agentic-ai-assistant
  4. Description: Capstone Project - Agentic AI Personal Assistant
  5. Set to: PUBLIC (important — must be public for submission)
  6. Check: ✅ Add a README file
  7. Click "Create repository"

STEP 3 — Upload Your Files
  Method A: Upload via Website (Easiest)
  ----------------------------------------
  1. Open your repository on GitHub
  2. Click "Add file" → "Upload files"
  3. Drag all your project files into the box:
       - agent.py
       - tools.py
       - requirements.txt
       - .env.example
       - README.md
       - Project_Documentation.pdf
  4. Scroll down, write a commit message: "Initial project upload"
  5. Click "Commit changes"

  Method B: Using Git (Advanced)
  ----------------------------------------
  1. Install Git from: https://git-scm.com/downloads
  2. Open terminal/command prompt in your project folder
  3. Run these commands one by one:
       git init
       git add .
       git commit -m "Initial project upload"
       git branch -M main
       git remote add origin https://github.com/YOUR_USERNAME/agentic-ai-assistant.git
       git push -u origin main

STEP 4 — Get Your Repository Link
  1. Go to your repository page on GitHub
  2. Copy the URL from the browser address bar
     Example: https://github.com/yourname/agentic-ai-assistant
  3. This is the link you'll paste in the Google Form submission!

STEP 5 — Verify It's Public
  1. Open an incognito/private browser window
  2. Paste your GitHub link
  3. If you can see the files without logging in — you're good! ✅

=================================================================
  IMPORTANT: Do NOT upload your .env file with your actual API key!
  Only upload .env.example (which has placeholder text)
=================================================================
